Configurar Sincronização
Para que toda a equipa veja os mesmos dados em tempo real, precisas de criar uma base de dados gratuita no Supabase (demora 5 minutos, gratuito).
1
Vai a supabase.com → Start your project → cria conta gratuita
2
Cria um novo projeto → escolhe nome boxpt-dashboard → define password → região West EU
3
No painel do projeto vai a SQL Editor e corre este comando:
CREATE TABLE orcamentos (id bigint generated always as identity primary key, data jsonb not null, imported_at timestamptz default now(), imported_by text);
ALTER TABLE orcamentos ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON orcamentos FOR ALL USING (true) WITH CHECK (true);
4
Vai a Project Settings → API e copia o Project URL e a anon public key
Usar sem sincronização (apenas local) →
